# Отборочное задание

## Git
1. Создай аккаунт на GitHub/GitLab, если такового нет.
2. Создай репозиторий для проекта карьерной программы.
3. Сделай коммит с инициализацией проекта.

ВАЖНО! В процессе работы с проектом необходимо делать правильные коммиты. Правильный коммит — это атомарное, осмысленное изменение с понятным сообщением.

## NPM
Проинициализируй свой проект для работы.

### Модули
Создай 2 модуля в проекте:
1. config.js, который хранит appName и настройки проекта. 
2. logger.js, который инкапсулирует console.log и возвращает функцию-обертку, с помощью которой можно выводить отформатированное сообщение в консоль с названием приложения.

ВАЖНО! Избегай циклических зависимостей в проекте.

## Event Loop
1. Создай файл scheduler.js. Файл должен содержать инициализирующий скрипт, который выполняется синхронно и логирует факт запуска scheduler.js. Также файл должен содержать функцию для управления периодическими задачами с 3 аргументами name (название), interval (интервал запуска) и task (функция, которая должна выполняться).
2. Добавь задачу, которая каждые 10 секунд будет логировать слово «running».
